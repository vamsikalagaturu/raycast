# raycast

## depends

### libraries
- [raylib](https://www.raylib.com) - need to be installed on system
- [nob.h](https://www.github.com/tsoding/nob.h) - included

### os
- Linux x86_64

## build

### build the nob.h once

```bash
cc -o nob nob.c
```

### build the game

```bash
./nob
```

## run

```bash
./build/game
```
